# **Bootcamp: Desarrollo de Aplicaciones Full Stack Python Trainee V2.0**

Bienvenido al repositorio oficial del programa **Desarrollo de Aplicaciones Full Stack Python Trainee V2.0**. Esta es una iniciativa estratégica impulsada por **SENCE**, **Talento Digital para Chile** y **Sustantiva OTEC**.

Este bootcamp está diseñado para transformar el potencial técnico en capacidades profesionales, formando desarrolladores de software competentes en el ecosistema **Python** y **Django**. El enfoque se centra en la construcción de soluciones web escalables, la manipulación avanzada de bases de datos y la implementación de buenas prácticas de la industria.

Para una descripción técnica detallada de la arquitectura de este repositorio, consulta: **[DOCUMENTACION_PROYECTO.md](./DOCUMENTACION_PROYECTO.md)**.

-----

## **Propósito del Programa**

Desarrollar las habilidades necesarias para construir, mantener y desplegar piezas de software utilizando el lenguaje Python y sus frameworks asociados. El objetivo final es la creación de aplicaciones web *Full Stack* capaces de operar bases de datos complejas para resolver problemáticas reales de las organizaciones.

## **Stack Tecnológico Principal**

  * **Lenguajes:** Python 3.5+, JavaScript (ES6+), SQL.

  * **Bases de Datos:** PostgreSQL, MySQL, SQLite.
  * **Herramientas de Entorno:** Git, GitHub, Visual Studio Code, Pip, venv.
